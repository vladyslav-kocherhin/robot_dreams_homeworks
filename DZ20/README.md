# Lesson 20 - Cucumber.js

1. **What is BDD**
2. **Gherkin as a BDD syntax**
3. **Cucumber.js installation and configuration**

## How to Run the Code

To run the code, follow these steps:

1. Install the necessary dependencies:
    ```sh
    npm install
    ```

2. Run tests:
    ```sh
    npm run test
    ```
